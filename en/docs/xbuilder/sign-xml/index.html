Sign XML

To sign an XML you need to have two objects:

  • X509Certificate and
  • PrivateKey

You can load those objects with the methods and code you create convenient.

Load certificate#

Using .pfx#

You can load X509Certificate and PrivateKey from a .pfx file using:

InputStream ksInputStream = new FileInputStream(new File("myCertificate.pfx"));
CertificateDetails certificate = CertificateDetailsFactory.create(ksInputStream, "myCertificatePassword");
X509Certificate certificate = certificate.getX509Certificate();
PrivateKey privateKey = certificate.getPrivateKey();

Sign XML#

String xml; // See docs to create an XML
String signatureID = "mySignID"; // Your Signature ID
// Get your certificate using the method of your preference
X509Certificate certificate;
PrivateKey privateKey;
Document signedXML = XMLSigner.signXML(xml, signatureID, certificate, privateKey);

That's it, you already have your XML signed!

tip

signedXML is an object of class org.w3c.dom.Document therefore you should not use System.out.println to view its contents; try to write the document to the hard disk.

tip

signatureID Is the value of ds: Signature Id="mySignID " inside the XML, for example:

<ext:UBLExtensions>
<ext:UBLExtension>
<ext:ExtensionContent>
<ds:Signature Id="mySignID" xmlns:ds="http://www.w3.org/2000/09/xmldsig#">
</ds:Signature>
</ext:ExtensionContent>
</ext:UBLExtension>
</ext:UBLExtensions>
