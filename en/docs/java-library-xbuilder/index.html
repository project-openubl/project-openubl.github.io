XBuilder

Java library use for:

  • Create XMLs.
  • Sign XMLs.

Requirements#

The only requirement is that you application use JDK8 or superior.

Resources#

  • Documentation
  • Videos

Where can I use XBuilder?#

Some of the most common integration are:

  • Desktop apps.
  • Web apps made on JavaEE and Jakarta EE. For instance Wildfly, JBoss, Glashfish, Tomcat, etc.
  • Web apps made on Spring and Spring Boot.
