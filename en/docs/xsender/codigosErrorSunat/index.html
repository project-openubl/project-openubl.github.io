Sunat error codes

Catalogue of errors in the Electronic Billing. This works for the SUNAT and the OSE. The codes have been classified according to the type of error:

  • From 0100 to 1999 Exceptions .-Serious errors that make file processing impossible. In these cases, the document is considered as not being reported, and the issuer must correct the problem to resubmit the document.
  • From 2000 to 3999 Errors that generate rejection.-In these cases the electronic document was processed, but errors were detected that do not allow to register it as a valid document.
  • From 4000 onwards Observations.-They are errors that do not invalidate the document and therefore the system will record the voucher as valid. The observations will be reported in the Reception Constancy.
