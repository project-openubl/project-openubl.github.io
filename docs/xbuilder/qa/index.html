El XML firmado es obtenido en un objeto de la forma:

NÃ³tese que el XML firmado no es un String sino un org.w3c.dom.Document por lo tanto no debes de intentar imprimirlo usando System.out.println(signedXML). Lo que debes de hacer es intentar escribirlo en un disco duro o convertirlo a bytes[] para que puedas empezar a usarlo.

TransformerFactory transformerFactory = TransformerFactory.newInstance();
Transformer transformer = transformerFactory.newTransformer();
transformer.transform(source, resultXml);
File file = new File("D:/sunat/operaciones/12345678959-01-F001-00000001.xml"); // Este archivo contiene el XML firmado
